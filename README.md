### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。

### 三、功能描述
系统分为（身份）：普通用户、仓库员工、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 个人中心
4. 修改密码
5. 公告查看
6. 物资查看
7. 物资申请

**仓库员工功能：**
1. 登录
2. 注册
3. 个人中心
4. 修改密码
5. 公告查看
6. 物资查看
7. 物资申请

**管理员功能：**
1. 登录
2. 个人中心
3. 修改密码
4. 管理员管理
5. 基础数据管理（仓库管理、公告类型管理、物资类型管理、物资申请类型管理）
6. 公告管理
7. 物资管理
8. 物资申请管理
9. 用户管理
10. 仓库管理员管理
